# Добавление компонентов, шаблонов, ассетов и анимаций (new-asset)

Используй этот документ, когда запрос требует добавить UI‑компонент, шаблон, ассет или анимацию.

## Компоненты

1. Определи область использования:
   - локальный для приложения/версии — размещай в `src/web/<system|user>/<AppName>/vN/widgets/`;
   - общий для нескольких приложений — размещай в `src/ui`.
2. Применяй токены и типографику проекта, не создавай дубли.
3. Обнови экраны, где компонент используется.

## Шаблоны

- Добавляй переиспользуемые паттерны в `src/templates`.
- Название файла отражает задачу (например, `SwipeDelete.tsx`).
- Шаблон должен быть нейтральным и пригодным для переиспользования.

## Ассеты

- Иконки: `src/assets/icons`.
- Иконки приложений: `src/assets/app_icons`.
- Шрифты: `src/assets/fonts` + обновить `src/styles/fonts.scss`.
- Используй относительные импорты из компонентов/экранов.

## Анимации

- Храни анимации в `src/animate`.
- При добавлении новой анимации обновляй `src/animate/README.md`.

## После добавления

- Если изменения затрагивают экраны, выполни синхронизацию по `sync-web-flutter.md`.
